Материалы

Добавляется по 2 стороны там, где указано «+ фанера».

Пока справочно, раскладку по листу не считаем.

Рейки: сечение 0.015 × 0.03 м. Для расчёта длины вычитаем ширину 0.03 м.

Фанера (раскрой деталей)

Лицо = (L + 2*фанера + 2*рейка) × (H + 2*фанера + 2*рейка). Бок = H × T. Крышка = (L + 2*фанера + 2*рейка) × (T).

Рейки (длины)

На каждую сторону: 4 длинные по размеру стороны + 6 коротких по размеру стороны минус 2×0.03 м.

Формулы с подстановкой
